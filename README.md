# The AI Automation Company

A React and Vite marketing site for The AI Automation Company, a service business focused on practical AI automation for small teams. The site presents services, workshops, customer work, company positioning, pricing, and contact paths for discovery calls.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router

## Getting Started

Install dependencies:

```sh
npm install
```

Start the local development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Preview the production build locally:

```sh
npm run preview
```

Run linting:

```sh
npm run lint
```

## Project Structure

- `src/App.tsx` defines the app routes.
- `src/pages/` contains the main site pages.
- `src/components/` contains shared layout and marketing components.
- `src/index.css` contains global styles and Tailwind setup.

## Routes

- `/` - Home page with hero, service highlights, pricing, and call to action.
- `/services` - Automation, n8n, AI agent, support, sales, data, and back-office services.
- `/workshops` - Training and workshop offerings.
- `/customers` - Customer-focused content and proof points.
- `/about` - Company story, values, and differentiators.
- `/contact` - Contact and discovery call path.
